# Project 3: Spots

### Overview

- Intro
- Figma
- Images

**Intro**

Spots, an interactive image-sharing application. Users will be able to add and remove photos, like photos from other users, and make minor adjustments to their profile. This project is made so all the elements are displayed correctly on popular screen sizes.

**Tech Stack**

- HTML
- CSS
- Responsive Design
